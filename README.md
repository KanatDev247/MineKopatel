# ⛏ MineKopatel

Minecraft-подобная воксельная игра в **одном HTML-файле**. Без сборки, без зависимостей, работает офлайн — просто откройте `index.html` в браузере.

🎮 **Играть:** https://kanatdev247.github.io/MineKopatel/
*(если ссылка не открывается — включите GitHub Pages: Settings → Pages → branch `main`, folder `/ (root)`)*

![screenshot](screenshot.png)

## Возможности (v2)

- **Бесконечный мир** — процедурная генерация (fbm-шум), чанки подгружаются на лету
- **Пещеры и данжи** — спагетти-пещеры, золотая руда, подземные сокровищницы и мшистые лабиринты с золотом
- **Выживание** — здоровье ❤️ и голод 🍗, урон от падения, утопление, регенерация при сытости, экран смерти и возрождение
- **Инструменты** — меч, кирка, топор, лопата: разные блоки ломаются с разной скоростью нужным инструментом, у меча урон по животным
- **Животные** — коровы, свиньи, овцы, курицы: детальные модели, живые анимации (виляют хвостами, клюют землю, машут крыльями), убегают от опасности, дают мясо 🍖 (клавиша **F** — съесть)
- **Графика** — мягкие тени, день/ночь с луной и звёздами, частицы при ломании блоков, рука с инструментом от первого лица, туман, анимированная вода, облака
- **Игровое время** — часы, счётчик дней и времени в игре
- **Мобильные устройства** — виртуальный джойстик, кнопки прыжка/ломания/установки, управление взглядом свайпом
- **Сохранение** — изменения мира сохраняются в localStorage

## Управление

| Действие | ПК | Телефон |
|---|---|---|
| Движение | WASD (Shift — бег) | джойстик |
| Прыжок | Пробел | кнопка ⬆ |
| Ломать | ЛКМ (удерживать) | кнопка ⛏ |
| Поставить блок | ПКМ | кнопка ▦ |
| Съесть мясо | F | кнопка 🍖 |
| Выбор слота | 1–9 / колесо | тап по слоту |

## Технологии

- [Three.js](https://threejs.org/) r160 (встроен в файл)
- Чанковый рендер с face culling и per-vertex ambient occlusion
- Процедурный texture atlas на Canvas (без внешних файлов)
- Кастомный шейдер неба (солнце, луна, звёзды, градиент)
